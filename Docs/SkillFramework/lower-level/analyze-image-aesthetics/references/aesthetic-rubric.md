# Aesthetic Rubric

## Blocker Scan

Before any scoring, check for:

- clipping or cropped content
- overlap or accidental collisions
- unreadable text
- elements hugging the frame edge without intent
- visual dead zones that make the composition feel broken
- browser chrome dominating a capture that is supposed to review page content

If any blocker is present, call it out before discussing polish.

## Score Bands

- `0-3`: broken or misleading
- `4-6`: weak, unstable, or not ready to present
- `7-8`: solid and usable
- `9-10`: unusually strong and deliberate

## Required Dimensions

| Dimension | Question to Answer |
| --- | --- |
| `空间美感` | Does the frame feel intentionally spaced, breathable, and visually calm? |
| `视觉层级` | Can the eye find the main point within 2 seconds? |
| `构图平衡` | Does the mass feel balanced, or is the frame lopsided without purpose? |
| `信息密度` | Is the page appropriately dense, not empty and not overloaded? |
| `可读性` | Can the important text and shapes be understood at presentation distance? |
| `演讲适配度` | Would this frame still support a live explanation on a PPT screen? |

## Evidence Rule

Every score should point to a visible cause such as:

- a crowded top-right cluster
- a receiver plane that is too small
- low contrast text
- an oversized empty lane
- competing highlight colors

Avoid empty labels like "composition is a bit off" unless you immediately name the visual cause.
