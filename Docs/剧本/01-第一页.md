# 第 01 页

## 页面目标

用最简单的 `Input -> f(x) -> Output` 建立整场 talk 的起点。这一页先给观众一个稳定、可记忆的最小骨架，后续所有更复杂的结构都从这个骨架演化出来。

## 页面改动表

| 项目 | 内容 |
| --- | --- |
| 页面标题 | Input -> f(x) -> Output |
| 页面副标题 | 最小公式骨架 |
| 当前页核心信息 | 输入经过一个函数，得到输出 |
| 与上一页关系 | 新起点，无需继承旧剧本 |
| 这一页新增了什么 | 一个水平主轴上的 `Input`、`f(x)`、`Output` 三点结构 |
| 这一页删除了什么 | 占位说明文案、待编写提示、无关卡片信息 |
| 这一页保留了什么 | 轻量背景、单一舞台框、稳定居中的舞台结构 |
| 需要强调的元素 | 中间的 `f(x)`，它是全页唯一主焦点 |
| 暂不实现 / 暂不出现 | 入场动画、镜头推进、额外注释框、更多节点 |
