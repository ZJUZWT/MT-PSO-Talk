## 页面目标

在保留第三页底部主轴和上方输入分区的前提下，把第 04 页改成真正的 Vulkan 语义：上方输入先进入 `Description`，再 create 出 `PSO`，真正高亮的 `GFXAPI CALL` 只剩下 `PSO -> GPU`。

## 页面改动表

| 项目 | 内容 |
| --- | --- |
| 页面标题 | Vulkan PSO |
| 页面副标题 | 先写进 Description，再创建出 PSO |
| 当前页核心信息 | `SPIR-V`、`Depth`、`Blend` 不再直接高亮地敲到 `GPU`，而是先进入 `Description`，再由橙色的 create API 生成 `PSO`，最后还剩 `PSO -> GPU` 这一条高亮运行时调用 |
| 与上一页关系 | 继承第三页的上下两层，只在中间插入 `Description / PSO` 这条创建语义，不重做整体骨架 |
